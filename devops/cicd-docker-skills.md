# DevOps & Cloud Copilot Skills
1. **GitHub Actions YAML**: `// Workflow to run Maven build and push to GHCR on main branch push` -> Generates valid `.github/workflows` YAML.
2. **Terraform/Bicep Auto-complete**: Start typing `resource "aws_instance"` -> Copilot fills required boilerplate.
3. **Kubernetes Manifests**: `// K8s deployment for nginx with 3 replicas and resource limits` -> Outputs complete `deployment.yaml`.
4. **Octopus Deploy Extension**: Query deployment status directly from IDE chat.
